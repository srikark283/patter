How it works

Three steps to never typing again

01

Press your hotkey

Anywhere on your desktop — F12 by default, or hold a key push-to-talk style. A minimal HUD shows you’re live.

02

Say it

Whisper or Parakeet transcribes on your GPU; your local LLM polishes the result using your memory and app profile.

03

It’s typed

Text appears at your cursor in whatever app you’re in. Every dictation lands in History, editable and searchable.

FAQ

Questions people actually ask

Does it really work with the internet off?

Yes. After the one-time model download, every stage — voice detection, transcription, LLM cleanup, meeting summaries — runs from your disk. Airplane mode is a fine place to dictate.

What hardware do I need?

macOS 11+ on Apple Silicon, or Windows 10+ on x64. Whisper Tiny runs happily on 8 GB of RAM; Parakeet and Large v3 Turbo like 16 GB. Transcription uses the GPU (Metal on Mac), so your fans stay quiet.

Which languages are supported?

English on the small Whisper models and Parakeet v2. Parakeet v3 covers 25 European languages; Whisper Large v3 Turbo handles 99+. Pick in Settings — the table above has the trade-offs.

Do I need Ollama?

Only for AI cleanup, memory, and meeting summaries. Without it, Patter still transcribes and types raw text perfectly well. Install Ollama later and the extra features light up.

macOS says the app is "damaged" — why?

Patter is an indie app without Apple's $99/yr notarization, so macOS quarantines the DMG download. The one-line curl installer avoids the flag entirely; or run xattr -cr /Applications/patter.app once. The code is open — audit it, or build it yourself.

Where is my data stored?

A SQLite database and model files in your user data folder. History, meetings, memory — all plain local files you can inspect, export, or delete. No account exists to leak.

Is it really free?

MIT-licensed, no tiers, no trial, no telemetry. It's a tool, not a funnel.

Install

One command, no dialog drama

Terminal
curl -fsSL https://raw.githubusercontent.com/srikark283/patter/main/install.sh | sh

Installs the latest release to /Applications. Then launch from Spotlight — onboarding walks you through mic access, a speech model, and your hotkey.

Prefer downloading the DMG yourself?
Grab it from the latest release, drag Patter to Applications, then clear the browser's quarantine flag — Patter is an indie app without Apple notarization, so macOS calls the download "damaged" until you run xattr -cr /Applications/patter.app. The curl route skips this entirely. First install only.
Prefer downloading the installer yourself?
Grab the .exe or .msi from the latest release and run it. Patter isn't EV-signed, so Windows SmartScreen may show "Windows protected your PC" — click More info → Run anyway. The PowerShell route skips this entirely. First install only.
