Patter is local-first dictation and meeting notes. Press a hotkey, talk, and polished text lands wherever your cursor is — transcribed and cleaned up entirely on-device.

Apple Silicon · macOS 11+ · Free & open source · Auto-updates

Private by design. Whisper and Parakeet run on your Mac's GPU. The LLM is your local Ollama. The only network traffic is downloading models you explicitly ask for.

Install in a minute

One command — downloads the latest release and installs it:

curl -fsSL https://raw.githubusercontent.com/srikark283/patter/main/install.sh | sh

Then launch from Spotlight. Onboarding walks you through mic access, a speech model, and your hotkey. Updates install themselves in-app afterwards.

Prefer the DMG?

1

Download the DMG from the latest release and drag Patter to Applications.

2

Clear the quarantine flag — browser downloads get tagged by macOS, and Patter isn't notarized (indie app, no Apple Developer program), so it reads "damaged" until you run:
xattr -cr /Applications/patter.app

3

Launch. Same onboarding, same app — the curl route just skips step 2.
